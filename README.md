To work with this project, you need to understand where the all the files are.
-.osp - it's a setting to be seen on OSPanel. Uses MySql-8.4, PHP-8.1 and Apache.
-config - folder with a database connection file.
-include - folder with files wich are needed to be included into a php file as a part of an html, like header.
-logic - folder with files which represent logic of each function of site.
-styles - there are .css files that stlye.

All pages are located in main folder.
